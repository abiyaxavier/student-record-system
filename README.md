
# Student Record System in C

A simple **Student Record System** written in **C** using file handling.  
This program allows you to **add, view, search, update, and delete** student records.

## Features

- Add Student (Roll, Name, Marks)
- View all Students in a table format
- Search Student by Roll number
- Update Student record by Roll number
- Delete Student record by Roll number
- Data is stored permanently using a file (`students.txt`)

## How to Run

1. Open terminal or command prompt in your project folder.
2. Compile the program:

```bash
gcc student_record.c -o student_record.exe
